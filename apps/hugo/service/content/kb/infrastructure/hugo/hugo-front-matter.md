# Overview

Front matter is the metadata section at the beginning of every Hugo content file.

It defines the variables Hugo uses to organize, render, and build pages.

Understanding front matter is one of the most important concepts in Hugo because nearly every other feature builds on it.

<!--more-->

# The Big Picture

![Hugo Front Matter vs Terraform Workflow](/images/hugo-front-matters-vs-terraform-guide.png)

The diagram above compares Hugo's rendering pipeline with Terraform's configuration model.

Although they solve different problems, both tools follow a similar engineering pattern:

- Define structured inputs.
- Pass them through an engine.
- Produce consistent output.
